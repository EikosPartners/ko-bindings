Module: clickOff

A knockout binding that is used to allow detection of clicking on another element i.e. "clicking off"
Parameters:
Name Type Description
handler function the function that is called when click off
includes Array.<string> | Array.<HTMLElement> an array of class names or html dom elements that when clicked on will invoke the handler
excludes Array.<string> | Array.<HTMLElement> an array of class names or html dom elements that when clicked on will *not* invoke the handler
Source:
  • clickoff.js, line 7
Examples
return {
  clickOff: function() {
    alert('it works!');
  }
};
clickOff: {
   handler: function ( ) {
       alert('it works!');
   },
   includes: ['clickOn', 'mainContent'],
   excludes: ['clickOff', 'titleBar']
}
